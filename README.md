# AI Desktop Agent Kit

Turn any AI agent into a full desktop automation powerhouse with mouse control, keyboard input, screenshot analysis, and visual element detection.

## Features

- Mouse Control - Move, click, drag anywhere on screen
- Keyboard Control - Type text, press keys, shortcuts
- Screenshots - Capture and analyze screen content
- Image Recognition - Find elements by matching images
- Persistent Memory - AI remembers across sessions
- Working Indicator - See when AI is actively working

## Quick Install

```bash
git clone https://github.com/YOUR_USERNAME/ai-desktop-agent.git ~/ai-desktop-agent
cd ~/ai-desktop-agent
chmod +x setup.sh
./setup.sh
```
